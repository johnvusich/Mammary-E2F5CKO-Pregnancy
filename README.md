# Mammary-E2F5CKO-Pregnancy
This repository contains the data and analysis code for the manuscript "Mammary Alveolar Proliferation and Differentiation During Early Pregnancy is Regulated by E2F5".
## Citation

Mammary Alveolar Proliferation and Differentiation During Early Pregnancy is Regulated by E2F5
John Vusich, Jesus Garcia-Lerena, Ryan Corfixsen, Rawan Talib, Morgan Atkins, Aishwarya V. Bhurke, Ripla Arora, Eran Andrechek
bioRxiv 2024.11.27.625731; doi: https://doi.org/10.1101/2024.11.27.625731

## Repository Contents

- **params/**: Contains samplesheet files, configs, and a README.md file explaining the custom genome reference for RNA-seq
- **workflow/**: Includes all scripts for Nextflow pipeline execution
- **analysis/**: Includes all scripts for data processing, analysis, and visualization.
- **LICENSE**: License information for the repository.

## Usage

Detailed instructions on how to reproduce the analysis and use the data can be found in the respective directories. For any questions, please contact the lead author.
